Grid

1

Container

Containers provide a means to
center and horizontally pad your
site’s contents. Use .container for a responsive pixel width or .container fluid for width: 100% across all
viewport and device sizes.

2

Rows

Rows are wrappers for columns. Each column has horizontal padding (called a gutter) for controlling the space between them. This padding is then counteracted on the rows with negative margins.

3

Cols

Column classes indicate the number of columns you’d like to use out of the possible 12 per row. So, if you want three equal-width columns across, you can use .col-4.

4

Content

In a grid layout, content must be placed within columns and only columns may be immediate children of rows.
